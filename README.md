Client Management System (C++)

A console-based Client Management System developed using C++ to manage client records through file handling and vector data structures.

Features
Display all clients information.
Add new clients.
Update existing client information.
Delete clients.
Search and display client details.
Store and retrieve client data using text files.
Prevent duplicate account numbers.
Simple menu-driven interface.
Technologies Used
C++
STL (vector, string)
File Handling (fstream)
Console Application
Project Concepts

This project applies several C++ programming concepts, including:

Structures (struct)
Enumerations (enum)
Functions
Vectors
File input/output
String manipulation
Basic input validation
Data Storage

Client information is stored in a text file using a custom record format, allowing the program to load, modify, and save client data.

Purpose

The goal of this project is to practice building a complete CRUD (Create, Read, Update, Delete) application using C++ fundamentals and improve understanding of data management and program organization.
